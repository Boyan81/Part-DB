debug()

debug(string  $type, string  $text, string  $file = '', string  $line = '', string  $method = '', boolean  $silent = true)

Add an Element to the Debug-Log

Parameters

string $type

@li The type of the log message, e.g. "error", "warning", "hint", ... @l1 You can use every string you want, but this types will be highlighted: @li "error", "warning", "temp" @li "temp" means "temporary", it will be printed bold, so it's easy to find your message in a large list of messages. Use this only temporary!
string $text

The message of your choice
string $file

The file where the error/whatever is located (use the PHP constant "FILE")
string $line

The line where the error/whatever is located (use the PHP constant "LINE")
string $method

The method/function where the error/whatever is located (use the PHP constant "METHOD")
boolean $silent

@li If true, this function will never throw an exception! @li This function should never disturb the normal operation, so you should always use the silent mode!

Throws

\Exception

if there was an error and "$silent == false"

Example:
code

debug('error', 'This is your Message...', FILE, LINE, METHOD); @endcode
note

Use always "METHOD", not "FUNCTION" ("METHOD" works in classes too, "FUNCTION" not!)
