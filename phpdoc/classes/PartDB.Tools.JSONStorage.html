getItem()

getItem(  $key, boolean  $throw_exception = true) : mixed|null

Returns the item for the given key.

If no item exists for the given key, by default a RuntimeException is thrown. Use $throw_exception to disable that.

Parameters

$key

string The key for which the item should be get.
boolean $throw_exception

Set this to false, if you dont want to get an exception thrown, if no item with the key exists. Instead null is returned.

Returns

mixed|null —

The item with the given key. Null if $throw_exception is false, and no item with this key was found.
