class SystemVersion

Class SystemVersion

A SystemVersion object represents a system version with the following attributes:

  • major version
  • minor version
  • update version
  • (release candidate number)
  • type ('stable' or 'unstable', depends on release candidate number)

A version string has this structure: "major.minor.update.[RC]" --> "#.#.#[.RC#]" (brackets means "optional", # stands for numbers)
class

SystemVersion
note

We will always use the format "#.#.#[.RC#]" for handling with version numbers! Also the filenames of update packages and their version descriptions uses that format! Only for displaying the version number in a HTML output, we use the format "#.#.# [RC#]" (space instead of dot).
par

Examples:

  • "0.2.3": stable version 0.2.3
  • "0.2.3.RC4": unstable version 0.2.3, release candidate 4
